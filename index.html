What is javascript?

  • JS is both programming and scripting language
  • used to set some actions to html

History of JS

  • Brendan Eich in 1995
  • 1997 - ECMAScript
  • current - ES7, usability - ES6

How to create JS?

  • Internal - script tag in body
  • External - create a new file with .js and link - script tag

Variables

  • var - global
  • let - local
  • const - only one time initialization

Output Statement

  • console.log() - used to print the output on console screen
  • alert() - popup message
  • DOM

Input Statement

  • prompt()
  • input tags

Operators

  • Unary
    • increment (++)
    • decrement (--)
  • Binary
    • Arithmetic -> +, -, *, /, %
    • Relational / Comparision -> >, <, >=, <=, ==, != , ===, !==
    • Logical -> &&, ||, !
    • Bitwise -> &, |, ^, >>, <<
    • Assignment -> =, +=, -=, *=, /=, %=
  • Ternary

Decision Making Statements

  • if
  • if-else
  • nested-if
  • else if
  • switch
  • nested switch

Looping statements

  • for - same as java syntax but instead of int need to use let
  • while - same as java syntax
  • do while - same as java syntax

Arrays

  • Collection of different data elements
  • using [] -> create / identify the array variables

    • Functions

  • used to create a group of code as single statement
  • function keyword to create a function

    • DOM

  • Document Object Model
  • Document - refers to html page / class
  • Object - to access anything from the html
  • Model - to set an action to html
  • Document class will have all predefined function of html usages
  • document object -> document.functionName()
  • onclick property - used to call the function in js from html page
  • use the onclick property only in our button tag or input tag

    • functions in DOM

  • document.getElementById() - to get the tag with id name from html page to js page
  • document.getElementByClassName() - to get the tag with class name from html page to js page
