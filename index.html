  4. arrays - [10,12.141,"Hello"]

  3. const - one time initialization

Input Statement

Operators

      Unary Operators
    1. ++
    2. --
      Binary Operators
    1. Arithmetic -> +, -, *, /, %
    2. Relational -> >, <, >=, <=, ==, !=, ===, !==

Decision Making Statements

Hints:

    for addition, both needs to be in same data types
  • 4 + 5 -> 9
  • 4 + 5.3 -> 9.3
  • "10" + 5 -> 105 (because string is higher in +)
    for multiplication, both needs to be in same data types
  • 4 * 5 -> 20
  • 1.2 * 2 -> 2.4
  • "10" * 5 -> 50 (because in *, number will have higher)
